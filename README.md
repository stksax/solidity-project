- 開發時間也在評估範圍內，時間計算為開啟題目信件時間至最後一個 Commit 的時間。請勿試圖修改 Commit 來縮短計量時長。
- 如遇到非開發問題（包括但不限於網路異常），請立即回信通報。
- 允許使用包含但不限於 Google、ChatGPT、Stake Overflow 等服務，然而面試時我們仍會期待您對於運作原理有所了解
- 允許升級外部套件，但不可新增外部套件（除非是因升級引入的必要套件）
